How NewsFlow is built

  • ⚛️
    Next.js · React · TypeScript · Tailwind
    Frontend framework, components, and styles.
  • ☁️
    AWS Amplify + DynamoDB
    Hosting for the website and storage for subscriber records.
  • 📰
    RSS aggregation + Local Sources
    Headlines from major publishers in 10 countries.
  • 🎙️
    Google Translate TTS · Microsoft Edge Neural TTS
    Two text-to-speech engines — instant playback on the site, premium voices for the video pipeline.
  • 📺
    FFmpeg
    Coming soon — vertical YouTube Shorts with a news-anchor avatar.
  • ✉️
    Resend
    Daily briefing emails. If readers want it on WhatsApp too, that's where we'll go next.
  • 🌐
    BigDataCloud reverse-geocoding
    "Use my location" without API keys.
  • 🤖
    OpenCode · Claude Code
    AI coding assistants that pair-built this whole thing alongside me.
