About @credo92 @vsrivastav-pm
WebGPU model runs 100% locally. Pick a model and press Start engine — or just type help below.
Heads up: requires a WebGPU-capable browser (desktop Chrome or Edge work best; Safari 17+/Firefox partial) with hardware acceleration on. The first load downloads the model weights (~0.7–1 GB) into your browser cache — after that it starts fast and the model reasons offline on your own GPU. It's a tiny model, so edits aren't always perfect. Try help, ls, cat index.js, or ask it to add a sum function to index.js.
